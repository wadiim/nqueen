# NQueen

A N-Queen Problem solver.

![nqueen](https://user-images.githubusercontent.com/33803413/74772139-bb09bd00-528f-11ea-9e54-0b8a627a6b0c.png)

## Usage

``` shell
nqueen [-h] [--pretty]
```

### Options

Option | Meaning |
--- | ---
`-h`, `--help` | Show help message and exit.
`--pretty` | Pretty-print the results.
